# E-Commerce Web Application

## Overview

This project is a responsive E-Commerce Web Application developed using Spring Boot, Thymeleaf, MySQL, and Bootstrap.

## Features

* User Registration
* User Login
* Product Listing
* Product Search
* Product Details
* Add Product
* Shopping Cart
* Checkout Process
* Order Management
* Responsive Design

## Technology Stack

* Java
* Spring Boot
* Spring Data JPA
* Thymeleaf
* MySQL
* Bootstrap 5

## Setup Instructions

1. Clone the repository.
2. Create a MySQL database named `ecommerce_db`.
3. Update database username and password in `application.properties`.
4. Run `EcommerceAppApplication.java`.
5. Open `http://localhost:8080`.
